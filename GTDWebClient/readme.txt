--------------------------------------------------------------
			ALMA GTD
--------------------------------------------------------------

-Le fichier AlmaGTD.war est une archive déployable sous tomcat dans le dossier webapps.
Une fois tomcat déployé, il suffit de se rendre a l'adresse http://localhost:8080/AlmaGTD pour accéder à l'application.

-Le dossier sources contient les sources du projet.

-Le fichier rapport-ALMAGTD.pdf est le rapport principal du projet.
-Le fichier rapportRMI.pdf est le rapport d'objets distribués.


-Le fichier soutenance.pdf est le pdf de la soutenance.

The artifact toodledo.jar must be downloaded here: http://github.com/downloads/lant/toodledo-java/toodledo-java-0.2.1.jar
